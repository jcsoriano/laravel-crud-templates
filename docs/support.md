## Security

If you discover a security vulnerability, please report it responsibly. Do not open a public GitHub issue.

[View Security Policy ‚Üí](https://github.com/jcsoriano/laravel-crud-templates/blob/main/SECURITY.md)

**To report a security issue:**
- Email: `hi@jcsoriano.com`
- Include a clear description, steps to reproduce, and potential impact
- We will acknowledge receipt within 72 hours and work on a fix
